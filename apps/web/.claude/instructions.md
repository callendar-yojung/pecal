# Claude Project Instructions

- 이 프로젝트는 일정관리 서비스 "celindor"다
- Next.js project with TypeScript, Tailwind CSS, NextAuth.js, MySQL, Biome, and next-intl for i18n
- white & black theme support
- multi-language support (Korean and English)
- main features: user authentication, dashboard with sidebar, landing page
